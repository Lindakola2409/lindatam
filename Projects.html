Chimken Crusade

With my partner Jacob Scott, we created a game one semester using Unity and C#

Top-down autoscrolling ‘bullet-hell’ featuring cute farm animals in a 16-bit art style, where the player can’t fight back. Your goal is to help the protag, Chimken, make it through a series of levels while protecting the golden egg.

Mario
  • Loaded and displayed game levels from text files
  • Implemented scroll camera that centered player and controlled camera movement while handling restrictions on camera movement and player boundaries.
  • Utilized acceleration and velocity for gravity simulation and jump mechanics
  • Enemy AI (Goomba) has spawner actors to manage dynamic enemy spawning and has distance-based spawning logic
Mario Kart
  • Implemented spring camera for a smoother transition
  • Employed a HeightMap class with 2D grid data to represent track elvation
  • Utilized cross product calculations to determine if the the enemy vehicle AI should turn left or right
  • Lap and place tracking were implemented using checkpoints by reading in a csv and checking if cars pass the checkpoint positions
  • Included music and sound effects with support for fading in and out
Zelda
  • Loading of levels with tiled backgrounds using tile mapping technique
  • Implemented enemy behavior with pathfinding using the A* algorithm
  • Utilized callbacks with functions and lambda for custom behavior on damage and death events
  • Collision detection for enemies and obstacles
  • AudioSystem for managing game sounds and music
Pong
  • Orchestrated a dynamic and responsive game loop architecture that handled input processing, game state updates, and output generation.
  • Employed physics principles to craft the ball's motion and manage collision detection.
  • Crafted conditional algorithms, enabling collision detection, ball movement, and responsive paddle behavior.
  • Managed resource allocation and memory using pointer techniques, handling elements like windows and renderers, and ensuring efficient deallocation.
Asteroids
  • Employed strategic conditional control to govern diverse game object behaviors.
  • Utilized vectors for precise laser targeting and directional alignment.
  • Skillfully integrated Euler method to dynamically update the ship's position with precision
  • Introduced captivating randomization techniques for initial asteroid placement, complemented by screen-wrapping mechanics.
Star Fox Tunnel

  • Rendered health bar and UI animations by creating a UIComponent
  • Designed and implemented a dynamic game mechanic allowing shooting a block to trigger a cascading destruction effect among neighboring blocks, utilizing a component-based architecture, recursive algorithms, and dynamic gameplay interactions.
Pac-Man

Portal
  • Parsed json files and used the information to create Voice Over Triggers to play voiceover lines, open doors, and load levels

Frogger
  • Loaded levels by parsing in text file inputs
